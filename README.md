# 🚗 Smart Traffic Analysis using YOLO

An end-to-end Computer Vision project that detects vehicles, tracks them across video frames, detects license plates, and estimates vehicle speed.

## 📌 Features

- 🚘 Vehicle Detection
- 🆔 Multi-Object Tracking (ByteTrack)
- 🔍 License Plate Detection
- ⚡ Estimated Vehicle Speed
- 🎥 Video Processing & Annotation

## 🛠️ Tech Stack

- Python
- YOLO (Ultralytics)
- ByteTrack
- OpenCV
- Roboflow
- Google Colab

## 📂 Project Workflow

Traffic Video
↓
Vehicle Detection (YOLO)
↓
Vehicle Tracking (ByteTrack)
↓
Vehicle Cropping
↓
License Plate Detection (Custom YOLO)
↓
Estimated Speed Calculation
↓
Annotated Output Video

## 📊 Results

The output video includes:
- Vehicle Detection
- Vehicle Tracking IDs
- License Plate Detection
- Estimated Speed for each tracked vehicle

## 🚀 Future Improvements

- More accurate speed estimation using camera calibration
